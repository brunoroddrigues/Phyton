## Como usar

1. **Clone o repositório**:
    ```bash
    git clone https://github.com/seu-usuario/exercicios-python.git
    ```

2. **Acesse o diretório** com os exercícios.

3. **Abra e execute os arquivos Python** para resolver os exercícios:
    ```bash
    python exercicio.py
    ```
